# 📝 TypeScript CLI ToDo App

A simple and interactive **Command-Line ToDo App** built with **TypeScript**.  

## 🚀 Features

- ➕ Add new tasks  
- 📋 View all tasks   
- 🗑️ Delete tasks  

## 📦 Tech Stack

- **TypeScript**
- **Node.js**
- `readline-sync` – for user input  
- `chalk` – for colored output  
- `figlet` – for banner text
